### 功能说明

* **For Mac
```bash
01. 单文件操作,可勾选自动打开日志
02. 多文件操作,拖拽多个文件同时解压
03. 自动检测TRTC调用Enter Room API事件,列表展示,鼠标左键单击弹出菜单
```
### 使用
* **For iOS
```bash
01. 解压SDK日志查看
02. 文本搜索
```
